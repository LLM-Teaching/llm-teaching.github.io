SynthEddix: AI-Powered Learning Material Generator

SynthEddix is an innovative application designed to make content creation effortless for educators and improve the learning experience for students. By using advanced Large Language Models, SynthEddix automatically generates high-quality learning materials, such as multiple-choice questions and fill-in-the-gap exercises, from existing resources, including PDFs and web links.

Project Aim

Our goal is to build a system that allows educators to easily curate and adapt study content with the help of LLMs. The tailored material generation is designed with flexibility and usability in mind. SynthEddix streamlines the creation of engaging and effective learning resources.

Features

  • AI-Generated Questions: Automatically generate multiple-choice and fill-in-the-gap questions from PDFs or web links using the Qwen3 model.
  • Teacher Interface: Manage classes, subjects, student progress, and materials with ease.
  • Student App: Access personalized learning materials.
  • PDF Integration: Upload, store, and retrieve PDFs for question generation.
  • Customizable Content: Choose the type of questions to generate (multiple-choice, fill-in-the-gaps, or both).
  • Secure User Management: Register, log in, and manage user accounts with password encryption and reset functionality.

How it works

  • Upload Content: Teachers can upload PDFs or provide web links as input.
  • Generate Questions: The AI processes the content and generates structured questions in JSON format.
  • Save to Database: Generated materials are stored in a database for later retrieval.
  • Access Materials: Students can access the materials through the Student App, while teachers can manage and update them via the Teacher Interface.

Who We Are

This project is currently a collaboration between Mathilda Buschmann, Cornelius Wolff, and Leonie Grafweg, in partnership with the German Research Center for Artificial Intelligence (DFKI). Our interdisciplinary team wants to bridge AI and education to create innovative learning tools.
